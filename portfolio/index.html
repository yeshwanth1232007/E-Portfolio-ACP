ACP Mini Project

2D Graphics Console Editor in C

A command-driven computer graphics software console developed in C. It employs memory-managed character matrices to simulate a raster canvas and uses fundamental graphics drawing algorithms to render pixels on a text screen.

DDA Algorithm: Renders lines using incremental steps and float rounding.
Midpoint Algorithm: Renders circles efficiently using 8-way symmetry.
Interactive Canvas: Holds a 25x80 char array rendering empty ('_') and plotted ('*') cells.
